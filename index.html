REACT Paint

React // Sass // Firebase // Responsive

A paint app built in React where users can draw on the canvas, change the colour of the brush and adjust the size of the brush. Users are also able to undo and redo any changes that they have made to the canvas. The app is also integrated with Firebase Realtime Database which allows the user to save their drawings as well as load it back onto the canvas at a later time to continue working on it.

Get Lit!

React // React Router // Sass // API // Firebase // Responsive

An app that help users decide on what book to read next using the Good Reads API. Users are able to search for books by genre and the results that come back are the most popular books in that specific genre. Firebase Realtime Database and Authentication is also integrated with the app which allows users to login, save their favourite books to the database, delete it and check off if the book is being read or has been read.

classiFurs

HTML // Sass // Javascript // RESTful API // Responsive

Looking for pets to adopt in your city? classiFurs is an online classifieds website that uses the PetFinder API to search for pets that have been put up for adoption. Users can search for a pet by type, gender, age and location. Clicking on a pet will provide the user with a more detailed look at the pet and their adoption information.
