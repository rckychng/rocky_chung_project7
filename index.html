REACT Paint

A paint app built in React where users can draw on the canvas, change the colour of the brush and adjust the size of the brush. Users are also able to undo and redo any changes that they have made to the canvas. The app is also integrated with Firebase Realtime Database which allows the user to save their drawings as well as load it back onto the canvas at a later time to continue working on it.

Get Lit!

Live GitHub
  • React
  • React Router
  • Sass
  • API
  • Firebase
  • Responsive

An app that help users decide on what book to read next using the Good Reads API. Users are able to search for books by genre and the results that come back are the most popular books in that specific genre. When a user selects a book a more detailed look of the book is diplayed that includes: the title, author, description, rating, number of pages, books similar to the selected book and a link to the Good Reads page of the selected book. The app is also integrated with Firebase Realtime Database and Authentication which allows users to save their favourite books to the database as well as delete it and check off that the book is currently being read or if the user has finshed reading the book.
